# Skill Roadmap

> 1·2학년 팀원을 프로젝트에 투입하기 위한 학습 로드맵입니다.  
> 모든 사람이 모든 기술을 깊게 알 필요는 없지만, 자기 역할에 필요한 기본기는 갖춰야 합니다.

---

## Common Track

모든 팀원이 공통으로 익혀야 할 것

### Entry

- GitHub 계정 생성
- Markdown 문서 작성
- 터미널 기본 명령어
- 프로젝트 README 읽기
- Issue 작성법
- 회의록 작성법

### Junior

- Git branch
- commit message
- Pull Request
- 기본 디버깅
- API 개념
- 데이터 흐름 이해

### Advanced

- 시스템 구조도 작성
- 기술문서 작성
- 테스트 로그 정리
- 발표자료 구성
- 프로젝트 리스크 분석
