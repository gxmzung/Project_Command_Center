# PROJECT COMMAND CENTER

> 이 문서는 현재 진행 중인 프로젝트를 한눈에 보기 위한 마스터 스케줄표입니다.  
> 단순 일정표가 아니라 프로젝트별 목적, 진행률, 필요 학과, 역할, 현재 상태를 필드화한 운영 대시보드입니다.

## 1. CityBrain

**분야:** 스마트캠퍼스 / 학생식당 데이터 플랫폼  
**목표:** 학생식당의 메뉴, 혼잡도, 잔여 수량, 만족도 데이터를 기반으로 학교 운영과 학생 편의를 연결하는 플랫폼 구축  
**진행률:** ███████░░░ 70%  
**상태:** 구현·수상 경험 있음 / 고도화 진행

**필요 학과 / 역할**

컴퓨터공학과 : 백엔드·앱·DB 개발  
정보통신공학과 : 서버·네트워크 구조  
경영학과 : 운영 모델·비즈니스 모델·설문 분석  
산업경영공학과 : 데이터 기반 운영 최적화  
디자인/미디어 계열 : UI·UX·브랜딩  

**현재 작업**

Backend API 정리  
Android 앱 연동  
관리자 페이지 개선  
데이터 시각화  
학교 적용 가능성 정리
